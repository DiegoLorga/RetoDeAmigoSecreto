# Amigo Secreto

Este proyecto es una aplicación web sencilla para organizar el juego de "Amigo Secreto". Permite agregar nombres de amigos, mostrarlos en una lista y realizar un sorteo aleatorio.

## Descripción

La aplicación permite:
- Agregar nombres de amigos a una lista.
- Visualizar la lista de amigos agregados.
- Sortear aleatoriamente un amigo secreto.

## Instalación y uso

1. **Descarga o clona el repositorio:**
   ```
   git clone <URL-del-repositorio>
   ```
2. **Abre la carpeta del proyecto en tu editor o navegador.**
3. **Ejecuta el archivo `index.html` en tu navegador favorito.**

No se requieren dependencias externas ni instalación adicional.

## Cómo funciona

- Escribe el nombre de un amigo en el campo de texto y haz clic en "Añadir".
- Los nombres se mostrarán en una lista.
- Haz clic en "Sortear amigo" para seleccionar aleatoriamente uno de los nombres agregados.

## Posibles problemas

- Si el campo de nombre está vacío, se mostrará una alerta.
- Si no hay amigos en la lista, el sorteo no se realizará y se mostrará una alerta.
